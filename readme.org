* Projeto
o objetivo desse projeto é criar um sistema basico de controle de treino com o publico alvo sendo personal treiner.
** Features desejada
- dashboard financeiro
- criar treinos, enviar para os alunos
- gerenciar cobranças, promoções
- marketing digital por email
